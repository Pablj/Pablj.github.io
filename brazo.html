Brazo Robótico con Visión Artificial

Proyecto desarrollado en 6º semestre de Ingeniería Eléctrica Electrónica

Análisis de Imagen

Proyecto desarrollado durante el 6º semestre de la carrera. Se implementó un sistema de visión utilizando una ESP32-CAM como servidor web para la captura y envío de imágenes. Las imágenes fueron procesadas en Python mediante las librerías NumPy y OpenCV. Se aplicó el algoritmo de detección de bordes Canny para identificar contornos del objeto. Posteriormente, se realizó un análisis de los puntos extremos (superior e inferior) con el objetivo de calcular una referencia en la columna base, permitiendo centrar el objeto y facilitar su manipulación mediante el brazo robótico.

Control de Servomotores

El control de posición de los servomotores se realizó mediante comparación entre la posición deseada y la posición detectada por el sistema de visión. Dependiendo del error calculado (positivo o negativo), se enviaban correcciones a través de peticiones HTTP hacia un servidor implementado en la ESP32-CAM. Se diseñó una única función para el envío de comandos, donde los servos que no requieren modificación reciben el valor -1, el cual es interpretado por el servidor como mantenimiento de la última posición. Este enfoque permitió simplificar la lógica de control y optimizar la comunicación.

Diseño Mecánico de Piezas

El diseño estructural del brazo robótico se basó en el concepto de un robot tipo sumo. Se modelaron un total de 9 piezas optimizadas para reducir el consumo de filamento y el tiempo de impresión 3D. El modelado se realizó en Autodesk Fusion 360, considerando restricciones mecánicas, transmisión de movimiento y acoplamiento con servomotores. Como mejora futura, se recomienda reducir algunos milímetros en la punta del mecanismo de transmisión para aumentar la precisión y minimizar holguras en el movimiento.

Resultados

El sistema logró identificar la posición del objeto en tiempo real y ajustar el movimiento del brazo robótico para tomarlo y reubicarlo correctamente. Se validó la integración entre visión artificial, comunicación HTTP y control de servomotores, logrando un sistema funcional de manipulación automatizada.

Mejoras y Trabajo Futuro

Aunque el sistema logró una manipulación funcional del objeto, se identificaron diversas oportunidades de mejora orientadas a optimizar precisión, estabilidad y suavidad de movimiento.

  • Implementación de un algoritmo de control PID para lograr movimientos más uniformes y reducir oscilaciones en la corrección de posición.
  • Integración de control cinemático directo e inverso para mejorar la planificación de trayectorias del brazo robótico.
  • Sustitución del control basado en peticiones HTTP por comunicación más eficiente (por ejemplo, WebSocket o protocolo serial optimizado) para reducir latencia.
  • Implementación de filtrado digital (media móvil o filtro de Kalman) en la detección de posición para disminuir ruido en la señal visual.
  • Optimización estructural del diseño mecánico para reducir holguras y mejorar precisión en el efector final.
