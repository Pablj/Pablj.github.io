BRAZO ROBÓTICO

Sistema de manipulación automatizada desarrollado en el 6º semestre de Ingeniería. Integra visión artificial con ESP32-CAM, procesamiento de imágenes en Python (OpenCV) y diseño mecánico impreso en 3D.

Hardware ESP32-CAM / Servos
Software Python / OpenCV
Mecánica Fusion 360 / 3D Print

Visión General

ARQUITECTURA DEL SISTEMA

El proyecto requirió la sincronización de tres dominios distintos: la captura y procesamiento visual de los objetos, el envío de correcciones por red y la actuación mecánica del efector final para lograr una manipulación precisa.

Procesamiento de Imágenes

ANÁLISIS DE IMAGEN Y CANNY

Se implementó un servidor web en la ESP32-CAM para la captura de imágenes. El procesamiento se realizó en Python (NumPy y OpenCV), aplicando el algoritmo de detección de bordes Canny para identificar contornos. Se analizan los puntos extremos para calcular la referencia en la columna base y centrar el objeto.

Estructura Física

DISEÑO CAD Y HARDWARE

El diseño estructural se basó en el concepto de un robot tipo sumo. Se modelaron 9 piezas en Autodesk Fusion 360, optimizadas para reducir el consumo de filamento PLA en la impresión 3D. El ensamblaje tomó en cuenta las restricciones mecánicas, el acoplamiento directo a los servomotores y la protección de la circuitería, implementando diodos para evitar que las corrientes inversas generadas por los motores dañaran el microcontrolador.

Evaluación

RESULTADOS Y TRABAJO FUTURO

El sistema logró identificar y reubicar objetos en tiempo real, validando la integración de visión, red y actuación. Sin embargo, se definieron los siguientes vectores de mejora técnica:

  • Controladores: Implementación de algoritmos PID para suavizar movimientos y reducir oscilaciones.
  • Cinemática: Integración de control cinemático inverso para una planificación de trayectorias precisa.
  • Protocolo de Red: Migración de HTTP POST a WebSockets o protocolo serial para minimizar la latencia.
  • Filtrado de Señal: Uso de filtros de Kalman o media móvil para reducir el ruido visual en la detección.
  • Optimización CAD: Reducción de holguras en la punta del mecanismo de transmisión para mayor exactitud.
