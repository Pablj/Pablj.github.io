En este apartado iré subiendo proyectos que desarrolle en mi tiempo libre o proyectos hechos en la universidad.

Brazo robótico de cinco grados de libertad

Desarrollo de un sistema de control para un brazo robótico de 5 GDL utilizando una ESP32-CAM para captura de imágenes. La imagen es enviada a un servidor mediante peticiones HTTP (GET/POST), donde se procesa con Python para análisis visual. El sistema integra visión artificial y control embebido para la toma de decisiones en tiempo real.

Microcontrolador de 8 bits con NVIC en VHDL

Diseño y descripción en VHDL de la arquitectura de un microcontrolador de 8 bits, incluyendo unidad de control, ALU, registros internos y una máquina de gestión de interrupciones (NVIC simplificada). El proyecto modela el comportamiento a nivel de hardware, permitiendo comprender el manejo de interrupciones, flujo de ejecución y control del sistema.
