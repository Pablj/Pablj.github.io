Simulación de Planificador Shortest Job First (SJF)

Sistemas Operativos en Tiempo Real – UNAM
Autores: Pablo Cruz Morales, Adrián Cohen Suárez, Rodrigo Yael Rubio Ortiz

Resumen

En este proyecto se desarrolla una simulación de un planificador de procesos basado en el algoritmo Shortest Job First (SJF). El objetivo es modelar su comportamiento en un entorno controlado, permitiendo analizar la ejecución de procesos con diferentes cargas de trabajo e introducir el manejo de quantums y almacenamiento de contexto.

Objetivo

Comprender el funcionamiento de los sistemas operativos mediante la simulación de un planificador de procesos, analizando la gestión del CPU, el uso de quantums y la recuperación del contexto de ejecución.

Introducción

El planificador (scheduler) es una de las funciones fundamentales de un sistema operativo, ya que decide qué proceso obtiene el uso del CPU. Existen múltiples algoritmos de planificación, entre los más relevantes:

  • Round Robin
  • First Come First Served (FCFS)
  • Shortest Job First (SJF)
  • Priority Scheduling

En el algoritmo SJF se da prioridad al proceso con menor tiempo de ejecución, lo cual minimiza el tiempo promedio de espera.

Planteamiento del Problema

El proyecto parte del modelado básico del planificador SJF y posteriormente introduce modificaciones académicas, como el uso de quantums. Cuando un proceso no finaliza dentro de su quantum, su contexto debe ser almacenado para continuar su ejecución posteriormente.

Desarrollo

Se implementaron diez procesos que realizan operaciones vectoriales sobre arreglos de tamaño 1000. Cada proceso se mide en tiempo de ejecución para determinar su prioridad dentro del planificador.

Medición de tiempo de ejecución


for (int i = 0; i < 10; i++) {
    start = clock();
    procesos[i](A, B);
    end = clock();
    tiempos[i] = ((double)(end - start)) / CLOCKS_PER_SEC;
}

Asignación de Quantums


void asignar_quantums(int quantum) {
    datos.suma_vectores.quantum = quantum;
    datos.resta_vectores.quantum = quantum;
    datos.producto_vectores.quantum = quantum;
    datos.division_vectores.quantum = quantum;
}

Resultados Experimentales

La simulación permite observar cómo los procesos más cortos se ejecutan primero y cómo aquellos que exceden su quantum almacenan su contexto para continuar posteriormente.

Conclusiones

La simulación del planificador SJF con quantums permitió comprender de manera práctica conceptos clave de los sistemas operativos en tiempo real, como la planificación, la interrupción de procesos y la recuperación de contexto.
